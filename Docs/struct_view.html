Подробное описание

Класс, отвечающий за GUI и взаимодействие с пользователем.

Заметки
View не занимается рендерингом текста книги, однако хранит и использует текстуру, на которой рисуется текст и спрайт страницы.
См. также
Controller и SWText о рендеринге текста.

◆ SCALE

int View::SCALE = 2

Коэффициент, повышающий разрешение текста. По умолчанию равен 2.

Предупреждения
Этот коэффициент необходимо учитывать ВСЕГДА, когда вы работаете с позициями слов на странице / отслеживаете местонахождение мыши на странице / и т. д.
