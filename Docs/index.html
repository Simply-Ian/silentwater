Silent Water Reader

Silent Water – находящийся сейчас в разработке ридер fb2 с открытым исходным кодом

Использует следующие библиотеки и модули:

  • pugixml для парсинга FB2
  • libiconv для чтения файлов в отличных от UTF-8 кодировках
  • SFML как GUI-Background + отрисовка текста
  • TGUI для создания GUI
